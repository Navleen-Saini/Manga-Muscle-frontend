# Manga-Muscle-frontend

Manga Muscle is a workout-focused web application with an anime-inspired design. It provides a clean and interactive frontend interface connected to a backend system for handling data and user interactions.

Features
Structured workout-focused user interface
Search and navigation functionality
Interactive UI components (buttons, menus, icons)
Dynamic page rendering using Pug templates
Integrated with backend services for data handling
Tech Stack
HTML
CSS
JavaScript
Pug (templating engine)
Architecture

The project follows a frontend-backend separation:

Frontend (this repository): UI, templates, client-side logic
Backend (separate repository): API, authentication, database operations
